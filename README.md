# ⚽ UOS02057 Introduction to Football Data Analytics (2025 Fall)

<br>


## 📖 소개 (Repository Description)

본 리포지토리는 서울시립대학교 **'축구 데이터 분석'** 수업의 실습 튜토리얼 자료 소개를 위해 만들어졌습니다. 축구 경기 데이터를 활용하여 데이터 정제, 탐색적 데이터 분석(EDA), 시각화 및 모델링, 인공지능 모델링 등을 통해 의미 있는 인사이트를 도출하는 것을 목표로 합니다.

<br>

## 🛠️ 기술 스택 및 개발 환경 (Tech Stack)

* **언어 (Language):** Python
* **주요 라이브러리 (Libraries):**
    * `Pandas`: 데이터 핸들링 및 처리
    * `NumPy`: 수치 연산
    * `Matplotlib` & `Seaborn`: 데이터 시각화
    * `Scikit-learn`: 머신러닝 모델링
    * `Statsbombpy, mplsoccer` 등: 파이썬 기반 축구 데이터 분석 라이브러리
* **개발 환경 (IDE):** Jupyter Notebook, VS Code
